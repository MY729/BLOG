# 基础学习

## React脚手架初始化项目
```js
// 全局安装react的脚手架create-react-app
npm install -g create-react-app

// 创建项目
create-react-app my-react

// 进入项目目录，启动项目
cd my-react
npm start
```
打开链接预览 [localhost:3000](localhost:3000)

## 搭建页面
这里使用`Ant Design`组件库
```js
npm install antd --save
```
